Turtle Run 2020

Turtle Run is a family-friendly event that involves road running with participants taking part for their own enjoyment, especially nature lovers! Although Turtle Run is a fun run mainly focused on spreading awareness, we also have exciting prizes waiting for the more competitive lot who manage to come out on top.

About Us

Online activity registration system (a.k.a OARS) allows students to discover and register for popular events in residential colleges of University of Malaya (UM). Besides, students are able to report an issue should they found at residential college and order food from the cafeterias using OARS.
